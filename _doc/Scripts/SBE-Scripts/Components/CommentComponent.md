# CommentComponent

Any supported comments in scripts.

# Available syntax

## #[""] ##
Multiline comment

Sample:

{% highlight java %}

#["
    Example
"]
{% endhighlight %}

{% highlight java %}

#["Example"]
{% endhighlight %}
